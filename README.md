# Thietkehethongmangnoibo
Enterprise Network Simulation (Cisco Packet Tracer Student)
Overview

Thiết kế và triển khai hệ thống mạng doanh nghiệp mô phỏng với VLAN, routing, NAT và bảo mật.

Features
VLAN segmentation (HR, IT, Guest, Server)
Inter-VLAN Routing (Layer 3 Switch)
DHCP
NAT (Internet access)
ACL (Firewall cơ bản)
EtherChannel (LACP)
Testing
PC truy cập Internet thành công
Kiểm soát truy cập giữa các VLAN
Hệ thống vẫn hoạt động khi 1 link bị ngắt
